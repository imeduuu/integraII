Documentación de los tipos de Schema.org implementados

Organization

Dónde se usa: En el <Head> de las páginas principales (index, mapa, adopcion, donaciones, etc.)

Propósito:
Describe la organización “Huella Segura” para que Google y otros motores de búsqueda sepan que se trata de una entidad legal o corporativa.

Propiedades incluidas:

name: Nombre de la organización.

url: Página web oficial.

logo: Logo de la organización.

sameAs: Redes sociales oficiales (Facebook, Instagram).

contactPoint: Datos de contacto (teléfono, tipo de contacto, área servida, idioma disponible).
