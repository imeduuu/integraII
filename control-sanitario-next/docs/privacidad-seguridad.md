#  Políticas de Privacidad y Seguridad

Este documento describe cómo nuestra plataforma gestiona la información personal de los usuarios y las medidas adoptadas para garantizar su seguridad.  
Nuestro compromiso es proteger los datos de quienes interactúan con el sistema de adopciones, donaciones y registro de voluntarios.

---

##  1. Manejo de la información personal

Recopilamos y utilizamos datos personales únicamente para fines relacionados con la gestión de adopciones, registro de voluntarios y recepción de donaciones.  
La información recolectada puede incluir:

- Nombre completo del usuario.  
- Dirección de correo electrónico.  
- Datos de contacto (teléfono o ubicación aproximada).  
- Información voluntaria ingresada en formularios (por ejemplo, motivo de adopción o comentario en una solicitud).  

**Uso de los datos:**
- Procesar solicitudes de adopción y voluntariado.  
- Contactar al usuario para confirmar información o enviar notificaciones.  
- Mejorar la experiencia del usuario dentro del sitio.  

Los datos **no se comparten con terceros**, salvo requerimiento legal o consentimiento explícito del usuario.

---

##  2. Medidas de seguridad implementadas

Para proteger la información de los usuarios se aplican las siguientes medidas:

- **Conexiones seguras (HTTPS):** Todos los datos se transmiten de forma cifrada.  
- **Validación y saneamiento de datos:** Se emplean funciones de sanitización para evitar inyección de código o datos maliciosos.  
- **Control de acceso:** Solo usuarios autorizados pueden acceder a información sensible o secciones administrativas.  
- **Protección en base de datos:** Las contraseñas y datos personales se almacenan cifrados o en formato seguro (hash).  
- **Monitoreo de errores:** Se registran y controlan los intentos de acceso indebido o errores críticos del sistema.  

---

##  3. Recomendaciones y buenas prácticas

Para mantener la seguridad y privacidad de los datos, se recomienda:

- No compartir credenciales o contraseñas con terceros.  
- Utilizar contraseñas seguras (mínimo 8 caracteres, combinando letras, números y símbolos).  
- Evitar ingresar información sensible en redes o dispositivos públicos.  
- Mantener actualizado el navegador y antivirus.  
- Cerrar sesión al finalizar el uso del sistema.  

---

##  4. Preguntas frecuentes 

**1. ¿Quién tiene acceso a mis datos personales?**  
Solo el personal autorizado del sistema y los administradores que gestionan solicitudes de adopción o donaciones.

**2. ¿Puedo solicitar la eliminación de mis datos?**  
Sí. Puedes enviar una solicitud al correo de soporte para eliminar tu información de forma permanente.

**3. ¿Se comparten mis datos con otras organizaciones?**  
No. La información se usa exclusivamente dentro de nuestra plataforma.

**4. ¿Cómo protegen las contraseñas?**  
Todas las contraseñas se almacenan usando algoritmos de hash seguros y nunca en texto plano.

**5. ¿Qué hago si detecto un uso indebido de mis datos?**  
Contacta de inmediato al administrador o al correo de soporte especificado en la plataforma.
